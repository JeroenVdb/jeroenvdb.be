new-static-project
==================

My default setup for a new static project.

## Setup contains

- Jekyll (1.2.1)
- Grunt with configurated plugins:
    - contrib-concat
    - sass
    - jekyll
    - contrib-watch
    - contrib-uglify
    - contrib-jshint
    - bg-shell
    - imageoptim
    - contrib-copy
    - bless
- Basic SMACSS folder structure

## Run
- Clone this repository
- cd to the folder
- Run `npm install`
- Run `grunt`
- Site is served to `localhost:8000`
